API

Authentification

Ajoute le header Authorization: Bearer <API_KEY>.

Créer une instance

# curl example
curl -X POST "https://api.novahost.example/v1/instances" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name":"web-01","plan":"vps-pro","region":"fr-par"}'

# JS fetch
fetch('https://api.novahost.example/v1/instances',{method:'POST',headers:{'Authorization':'Bearer ' + API_KEY,'Content-Type':'application/json'},body:JSON.stringify({name:'web-01',plan:'vps-pro',region:'fr-par'})}).then(r=>r.json()).then(console.log)

Exemples de réponses

{
  "id":"inst_abc123",
  "name":"web-01",
  "status":"provisioning"
}
