API Reference NovaHost v1

REST API to manage instances, snapshots, floating IPs, and operations actions. All requests are JSON UTF-8 over HTTPS.

Base URL: https://api.novahost.dev/v1 Auth: Bearer Token Format: JSON

Endpoints Instances

GET/instances

List instances with pagination.

POST/instances

Create an instance from a plan and an image.

GET/instances/{id}

Retrieve instance details.

DELETE/instances/{id}

Permanently delete an instance.

Create an instance: examples

curl -X POST "https://api.novahost.dev/v1/instances" \
  -H "Authorization: Bearer NH_LIVE_xxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "ci-runner-01",
    "plan": "vps-pro-4",
    "region": "fr-par-1",
    "image": "ubuntu-24.04",
    "ssh_keys": ["infra_admin_key"],
    "tags": ["ci", "runner"]
  }'
const response = await fetch("https://api.novahost.dev/v1/instances", {
  method: "POST",
  headers: {
    "Authorization": "Bearer " + process.env.NH_API_TOKEN,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    name: "ci-runner-01",
    plan: "vps-pro-4",
    region: "fr-par-1",
    image: "ubuntu-24.04",
    ssh_keys: ["infra_admin_key"]
  })
});

const data = await response.json();
console.log(data.instance_id, data.status);
import requests

url = "https://api.novahost.dev/v1/instances"
headers = {
    "Authorization": "Bearer " + NH_API_TOKEN,
    "Content-Type": "application/json",
}
payload = {
    "name": "ci-runner-01",
    "plan": "vps-pro-4",
    "region": "fr-par-1",
    "image": "ubuntu-24.04",
    "ssh_keys": ["infra_admin_key"],
}

r = requests.post(url, headers=headers, json=payload, timeout=10)
print(r.json())

Errors API

Unified error format:

{
  "error": {
    "code": "quota_exceeded",
    "message": "Plan quota exceeded for region fr-par-1",
    "request_id": "req_5f0f8b2e",
    "details": {
      "resource": "instances",
      "limit": 20
    }
  }
}
HTTP Signification Recommended action
400 Invalid parameter Corriger le payload
401 Token invalide Regenerate API key
429 Rate limit exceeded Exponential retry with jitter
500+ Server error Retry + ouverture incident si persistant

Rate limits

Endpoints actions critiques

Response headers
X-RateLimit-Limit, X-RateLimit-Remaining, Retry-After
